# Numbers Game 

A simple web game where users try to guess a number or interact with numbers in engaging ways (depending on implementation). This project was built to practice interactive JavaScript logic, user input handling, and DOM updates.

**Live Demo:** https://kelseysemien.github.io/Numbers-Game/

---

## About

The **Numbers Game** is a beginner-friendly browser game that lets users play with numbers through clicking buttons and interactive feedback. This app demonstrates front-end logic, event handling, and dynamic HTML updates using JavaScript.

Still a work in progress, this project helps practice core web development skills and create a fun, interactive experience directly in the browser.

---

## Features

✔ Displays number updates based on user interactions  
✔ Uses JavaScript to process input and update the UI  
✔ Engaging feedback for each interaction  
✔ Clean and responsive layout

---

## Tech Stack

This project uses:

- **HTML5** – Page structure  
- **CSS3** – Visual styling  
- **JavaScript** – Game logic & UI interaction  
- Hosted on **GitHub Pages**

---

## How It Works

1. The main game logic is handled in JavaScript.  
2. User interacts with buttons or input elements.  
3. Script updates the displayed number or game state.  
4. UI responds dynamically to user actions.
